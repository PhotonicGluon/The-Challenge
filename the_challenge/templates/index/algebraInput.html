Information About Algebraic Inputs

All algebraic inputs are to be typed in the MathLex language. The specifics of the MathLex language can be found on this page. The following are several important pointers when entering algebraic expressions into the input field:

  • To enter the expression $ax$ you will need to enter the string a*x into the input field.
  • To enter a fraction like $\frac pq$ you will need to enter the string p/q into the input field.
  • To enter $\sqrt{ax+b}$ you will need to enter the string sqrt(a*x+b) into the input field.
  • To enter $\sec^2 \left(ax+b\right)$ you will need to enter the string (sec(a*x+b))**2 or (sec(a*x+b))^2 into the input field.
